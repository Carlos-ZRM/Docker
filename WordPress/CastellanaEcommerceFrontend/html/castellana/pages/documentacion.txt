En la carpeta estan distintas hojas con estilos especificos de las pantallas.
Se crearon los archivos syle.css que esta en la raíz del proyecto - menu.css - responsive.css / estos archivos son de suma importancia 
para que las vistas del sitio se vean de manera correcta visualmente tanto en desktop como en mobile.

Los archivos resguardados en la carpeta CSS traen estilos especifico que influyen en pantallas especificas del proyecto.

Los archivos que se encuentran en la carpeta Lib estan separados por carpetas internas que contienen los archivos y 
dependencias para que funcionen de manera correcta ciertas librerias.

Esta la carpeta del framework "Boostrap" la cual es de suma importancia para que el diseño del sitio responda de manera correcta en 
los distintos dispositivos como en las resoluciones de pantallas.

La carpeta datatable contiene los archivos del plugin "Data Table". Este influye para que las tablas que se utilizaron en el proyecto fluyan de manera correcta.

En la carpeta script contiene archivos .js que influyen para efectos, eventos y acciones que contiene el sitio.

El archivo slider-principal.js contiene codigo el cual sirve para la manupilación del slider de los productos, estos slider estan armados con componentes de dos plugin´s
1- Slick slider https://kenwheeler.github.io/slick/ 
2- Swiper slider https://idangero.us/swiper/
Para una mejor manipulación y mejor esperiencia como usuario y desarrollador se utilizaron las opciones de la documentación de estos
la mayor parte de estas opciones son boolean - javascript.

El archivo calcular trae los los datos que se pintan en la tabla de calcular evento, para sus funcionamiento de esta tabla se utilizo el plugin Data Table.

El archivo cooking.js sirve para el funcionamineto del efecto que se agrego en la pantalla de gourmet.

El archivo main.js trae extractos de codigo javascript que son fundamentales para varios eventos que se llevan acabo en el funcionamiento de la web.

El archivo maps.js trae la información integrada de las locaciones de las sucursales para su funcionamiento de este se utilizo el plugin Mapbox.

El archivo wow.min.js es un plugin que permite animación de entrada o salida de objetos que se agregaron a la web.

En la carpeta images / img estan agregados todas las imagenes con formato .jpg / .png / .svg

En la carpeta html estan agregados las platillas .html que construyen el sitio web
